# GraphQL Starter

A next-drupal Build frontend wed site via json api

## How to use

1. `git clone https://github.com/nhanlq/itconsultis-frontend.git`
2. Run `yarn install`
3. Run `yarn dev`


## Requirement
1. npm version 10.8.x
2. yarn version 1.22.x

## Noted
- Change drupal URL backend site in .env file.
